# Рамка с регистрами
## Описание
Это резидентная программа для **DOS**.
По нажатию клавиши "ё" она выводит на экран зелёную рамку с таблицей регистров (**AX, BX, CX, DX, ES, SS, CS, SI, DI, BP, SP, IP**) и флагов (**CF, ZF, SF, OF, PF, AF, IF, DF**).
Нажатие клавиши "=" выключает отображение и убирает рамку с экрана.

Предназначено для запуска в DOSBox.

## Сборка
Для компиляции используйте **Turbo Assembler (TASM)**:
```powershell
tasm regs.asm
tlink /t regs.obj
```

## Запуск программы

### Формат ввода
```powershell
regs.com <Символы, из которых будет состоять рамка (необязательно)>
```

#### Параметры командной строки
Вы можете изменить символы, используемые для отрисовки рамки, в следующем порядке:
```
1. Верхний левый угол (по умолчанию ╔ – 0xC9)
2. Горизонтальная линия (по умолчанию ═ – 0xCD)
3. Верхний правый угол (по умолчанию ╗ – 0xBB)
4. Вертикальная линия (по умолчанию ║ – 0xBA)
5. Нижний левый угол (по умолчанию ╚ – 0xC8)
6. Нижний правый угол (по умолчанию ╝ – 0xBC)
```

#### Примеры настройки рамки

##### Пример 1
Если запустить программу без аргументов командной строки и нажать "ё", она нарисует рамку, используя стандартный стиль границ.

```powershell

```

##### Пример 2
Если запустить программу с 1-5 аргументами командной строки и нажать "ё", она нарисует рамку, используя предоставленные пользователем символы и часть символов по умолчанию.

```powershell

```

##### Пример 3.1
Если запустить программу с 6 аргументами командной строки и нажать "ё", она нарисует рамку, используя только предоставленные пользователем символы.


Если запустить программу с более чем 6 аргументами командной строки и нажмете "ё", она все равно отобразит рамку, используя только первые 6 предоставленных пользователем символов, а остальные будут проигнорированы.

### Использование резидентной программы после установки

1. Запустите любую программу.
2. Нажмите "ё".
3. На экране появится зеленая рамка с текущими значениями регистров и флагов программы, которая была прервана нажатием клавиши. Данные обновляются при каждом тике таймера (~18,2 раза в секунду).
4. Чтобы скрыть рамку, нажмите `=`. Отображение регистров отключится, и рамка будет удалена с экрана.
5. Программа остаётся в памяти до перезагрузки.

#### Пример 1
Напишем простую программу с бесконечным циклом, чтобы проверить, как она работает.

```assembler
.model tiny
.code
org 100h

Start:

permanent:

mov bx, 02222h
mov cx, 03333h
mov dx, 04444h

mov ax, 05555h
push ax
pop ds

mov ax, 06666h
push ax
pop es

mov ax, 07777h
push ax
pop ss

mov si, 09999h
mov di, 0AAAAh
mov bp, 0BBBBh
mov sp, 0CCCCh

mov ax, 01111h
jmp permanent

end Start
```

Перед запуском этой тестовой программы установите резидентную программу и включите фрейм с регистрами (нажмите "ё"). Затем запустите тестовую программу.
Этот пример подтверждает, что резидентная программа корректно считывает и отображает все регистры.
